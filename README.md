# 小熊洗护 微信小程序介绍

小熊洗鞋洗衣 介绍
一个 本地化洗鞋，洗衣服务的微信小程序，有加盟店，支持自送 或者快递，具有业务分销功能。

#### 软件架构
微信原生小程序，已经在使用

![体验二维码](other%E4%BD%93%E9%AA%8C%E4%BA%8C%E7%BB%B4%E7%A0%81.jpg)


#### 安装教程

1.  克隆到本地
2.  微信小程序开发工具直接打开运行
3.  接口可以使用我的，在开发工具设置不校验合法域名，（用于学习）
